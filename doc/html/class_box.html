Detailed Description

This class represents a cube object in a three dimensional space. Is defined by two vectors that fix the diagonal of the cube. Can retreive information about area, volume and the centroid of the object. Can perform basic operations souch as translations.

const Vector3D Box::getCorner ( short  index) const

Calculates the coord of one corner of the cube object. The corners can be enumerated starting in the front left bottom corner from left to right, then up and finally to the back repeating the same enumerating proccess up to 8 corners. The minCorner is the index 0 and the max corner the index 7.

Parameters
shortindex : The index of the requested corner.
Returns
Vector3D : The first corner of the cube object.
Exceptions
IllegalArgumentExpetion: If (7 < index < 0) in souch case the given index does not represent a cube corner.
