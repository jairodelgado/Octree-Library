Detailed Description

This class is an abstract superclass to read a set of Geometries from a source. This source could be a file, a data base, or a resource at internet. Specific implementations should be provided in derived classes.

enum Reader::Readable

This enum defines the types of geometries that the reader should be able to load. This is an integer enum where the corresponding integer number represents the number of vectors that has to be loaded in order to create the specified geometry.
